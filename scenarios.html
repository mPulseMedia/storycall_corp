What are some sample exchanges with StoryCall?

The following are examples of what it’s like to use StoryCall to communicate with your grandma.

Jump to a scenario

  1. Kids successfully set up
    1. Successful setup
    2. Kid error: incorrect keyword
    3. Kid error: phone number sent as text
    4. Phone number audio is unclear / incomplete
    5. Change the call-to number
  2. Message exchange
    1. Audio question + audio reply
    2. Reschedule: call back in about an hour
    3. kid error: Kid sends text instead of audio
    4. No answer / voicemail / retry
  3. Texting for help

    2. HELP: how to attach a voice recording
    3. HELP: change the number
  4. Texting to stop
    1. STOP: opt out
    2. Restart after STOP

No problem. In your texting app:

- Tap the + or attachment button near the message box.

- Choose Audio / Voice / Record.

- Record your message (or the phone number), then tap Send.

Tip: avoid speech-to-text dictation — we need an actual audio recording.
